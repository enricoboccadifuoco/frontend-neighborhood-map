## Frontend Neighborhood map

### Getting started

#### How to run
    cd /path/to/frontend-neighborhood-map
    npm update
    bower update
    grunt

    python -m SimpleHTTPServer 8080

#### Grunt taks

1. uglify task

    ``` bash
    grunt uglify
    ```

1. concat task

    ``` bash
    grunt concat
    ```

1. jshint task

    ``` bash
    grunt jshint
    ```
